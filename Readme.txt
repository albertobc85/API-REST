
This project implements "Express", "Body Parser", "Nodemon". 
You can get more information about this libraries in https://expressjs.com/es/ https://www.npmjs.com/package/body-parser and https://www.npmjs.com/package/nodemon

Enter the next command to initialize the API REST server:
nodemon server.js

You can access to the JSON data by the next links:

http://localhost:3000/policies
http://localhost:3000/policies/:id
